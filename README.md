To-Do App


A simple, beginner-friendly To-Do app built with HTML, CSS, and JavaScript. Keep track of your daily tasks, notes, and routines.

Purpose


This project was created just to practice and strengthen my foundation in front-end development.

Features


Add, edit, and delete tasks 📝
Mark tasks as completed ✅
Saves tasks in your browser so they stay after refreshing 💾
Sections for daily notes, food routines, or reflections
